# Jogo da Forca

Esse jogo de terminal foi criado na aula prática do curso da Softblue.
Criei a classe FailedTry para visualizar a forca conforme os erros vão acumulando.

### Commads

Para fazer o build

`mvn clean package`

Para rodar o jogo

`java -jar target/hangman-1.0.0-SNAPSHOT.jar`
